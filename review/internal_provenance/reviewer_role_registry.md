# Reviewer Role Registry — Internal

- Public label `专家 X`: source review artifacts supplied by the project owner. The source archives do not disclose a specific generating model or tool.
- Public label `专家 Y`: reserved for a separately evidenced second expert review; not used for the current scientific judgments.
- `复核者`: deterministic repository integration and validation, not a new scientific adjudication.
- `Judge`: GPT-5.6-sol fixed-rubric scoring role.
