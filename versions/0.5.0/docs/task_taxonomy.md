# Task Taxonomy

## 中文

mini-benchmark 0.5.0 使用两条分类轴：领域知识和研发阶段。

### Domain

| Domain | What It Tests |
|---|---|
| organic_chemistry | 显色受体、聚合物基体、导电聚合物、溶剂和酸度效应 |
| physical_chemistry | 吸附、脱附、扩散、动力学、湿度竞争和响应/恢复权衡 |
| inorganic_chemistry | 金属氧化物、p/n 型响应、氧空位、贵金属敏化和硫化转化 |
| materials_science | 薄膜、孔结构、基底、电极、封装和批次差异 |
| general_chemistry | 氧化还原、酸碱、ppm、MFC 稀释、沉淀和混合气效应 |
| analytical_chemistry | LOD、校准、反射率、XRD、XPS、SEM/EDS、GC、Raman |
| technical_chemistry | 涂布、丝网印刷、煅烧、连续浸渍和小试放大 |
| toxicity_and_safety | 有毒气体、氧化剂、DMF、含银危废、纳米粉体和安全 gate |

### Scenario Stage

| Stage | What It Tests |
|---|---|
| 文献分析 | 从公开知识中提取可迁移机理和证据边界 |
| 实验设计 | 设计变量、对照、验证矩阵和决策规则 |
| 实验进行 | 识别涂布、配气、加热和测试腔上线中的操作约束 |
| 结果分析 | 诊断漂移、恢复、校准、表征解释和数据质量 |
| 下一步计划 | 形成路线取舍、产品 gate 和后续验证方案 |
| 安全边界 | 识别安全 gate、工程控制和合规边界 |

## English

mini-benchmark 0.5.0 uses two taxonomy axes: domain knowledge and R&D workflow stage.

### Domain

| Domain | What It Tests |
|---|---|
| organic_chemistry | Chromogenic receptors, polymer matrices, conducting polymers, solvent effects, and acidity |
| physical_chemistry | Adsorption, desorption, diffusion, kinetics, humidity competition, and response/recovery balance |
| inorganic_chemistry | Metal oxides, p/n-type response, oxygen vacancies, noble-metal sensitization, and sulfide conversion |
| materials_science | Thin films, porous structures, substrates, electrodes, packaging, and batch variation |
| general_chemistry | Redox, acid-base chemistry, gas ppm, MFC dilution, precipitation, and mixed-gas effects |
| analytical_chemistry | LOD, calibration, reflectance, XRD, XPS, SEM/EDS, GC, and Raman |
| technical_chemistry | Coating, screen printing, calcination, continuous impregnation, and pilot scale-up |
| toxicity_and_safety | Toxic gases, oxidizers, DMF, silver waste, nanopowders, and safety gates |

### Scenario Stage

| Stage | What It Tests |
|---|---|
| 文献分析 | Transferable mechanisms and evidence boundaries from public knowledge |
| 实验设计 | Variables, controls, validation matrices, and decision rules |
| 实验进行 | Operational constraints in coating, gas mixing, heating, and chamber commissioning |
| 结果分析 | Drift, recovery, calibration, characterization interpretation, and data quality |
| 下一步计划 | Route selection, product gates, and validation planning |
| 安全边界 | Safety gates, engineering controls, and compliance boundaries |
