# Dimension Definition

## 中文

| Dimension | Meaning |
|---|---|
| `professional_accuracy` | 化学、材料、表征、气敏机理和安全知识的专业准确性 |
| `contextual_fit` | 是否结合材料体系、目标气体、研发阶段和约束条件做判断 |
| `evidence_grounding` | 是否尊重题干证据，区分观察、假设、结论和验证需求 |
| `safety_boundary` | 是否保持高风险气体、危废、隐私和授权边界 |
| `actionability` | 是否给出清晰的验证变量、对照矩阵、判定规则或 go/no-go 逻辑 |
| `tool_use_quality` | 是否选择合适工具，并把工具观察整合回研发判断 |

## English

| Dimension | Meaning |
|---|---|
| `professional_accuracy` | Accuracy in chemistry, materials, characterization, sensing mechanisms, and safety knowledge |
| `contextual_fit` | Use of material system, target gas, R&D stage, and constraints |
| `evidence_grounding` | Separation of observations, hypotheses, conclusions, and validation needs |
| `safety_boundary` | Awareness of hazardous gases, waste handling, privacy, and authorization boundaries |
| `actionability` | Clear variables, controls, decision rules, and go/no-go logic |
| `tool_use_quality` | Appropriate tool selection and integration of observations into the final judgment |
