# 0.4.0 Summary

0.4.0 是 SGS152 之前的 Domain Core 版本。它证明了半导体气敏材料研发任务可以转成结构化 benchmark，但主集分数高分聚集，强模型区分度不足。

## 0.4.0 Results

| Model | 0.4.0 Main MCQ | Robustness |
|---|---:|---:|
| GPT-5.5 | 80 / 82 | 35 / 40 |
| MiMo v2.5 Pro | 80 / 82 | 36 / 40 |
| DeepSeek V4 Pro | 76 / 82 | 30 / 40 |

## 关键设计变化

- 0.4.0 主集以 Domain Core Set 为主，覆盖研发判断、实验设计、结果解释和安全边界。
- 0.4.0 robustness variants 用于测试相近题面下的判断一致性。
- 0.5.0 引入 Scientific Stress Set，用于观察科学规则、定量、谱图、结构性质和安全风险识别中的强模型边界。
- 0.5.0 修复 MCQ prompt 对 E 选项支持不足的问题。
- 0.5.0 把 active package 提升到根目录，历史版本保留为轻量摘要和 Git 历史。
